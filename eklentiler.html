Editör Eklentileri

OrkhonJS, tüm popüler editörler ve araçlar için kapsamlı eklenti desteği sunar

💜

VS Code Eklentisi

Sözdizimi renklendirme, IntelliSense otomatik tamamlama, LSP hata gösterimi, Debug Adapter ile hata ayıklama, 48+ kod parçacığı (snippet), görev sağlayıcı ve Web IDE desteği.

cd extensions/orkhonjs-ide
npm install
npm run compile
# F5 ile Extension Host'u başlatın
TextMate LSP Debug Adapter Snippets
🧡

JetBrains Eklentisi

IntelliJ IDEA, WebStorm, PhpStorm ve diğer JetBrains IDE'leri için. Sözdizimi renklendirme, kod tamamlama, canlı şablonlar, parantez eşleştirme ve çalıştırma yapılandırması.

cd extensions/jetbrains
./gradlew buildPlugin
# build/distributions/ → .zip dosyasını kurun
# Settings → Plugins → Install from Disk
IntelliJ WebStorm Live Templates
💚

Vim/Neovim Eklentisi

Lua tabanlı, tam özellikli Neovim eklentisi. Sözdizimi renklendirme, akıllı girinti, LuaSnip snippet'leri, LSP entegrasyonu, Türkçe komutlar (:OrkhonDerle, :OrkhonCalistir).

-- lazy.nvim ile:
{
    "orkhonjs/orkhonjs",
    config = function()
        require("orkhonjs").setup({
            auto_compile = true,
            snippets = true,
            lsp = true,
        })
    end,
    ft = "orkhon",
}
Neovim Lua LSP LuaSnip
🔍

Tarayıcı DevTools

Chrome/Edge tarayıcıları için OrkhonJS hata ayıklama paneli. Türkçe hata mesajları (50+ JS hatası çevirisi), JS→OrkhonJS kaynak çevirisi, AST görüntüleyici, performans izleme.

# Chrome:
# 1. chrome://extensions/ adresini açın
# 2. "Geliştirici modu" aktifleştirin
# 3. "Paketlenmemiş yükle" tıklayın
# 4. extensions/browser-devtools/ seçin
Chrome Edge Source Maps
⚙️

GitHub Actions

CI/CD pipeline'ınızda OrkhonJS lint ve derleme. Pull request'lerde otomatik derleme, sıkı mod kontrolü, özelleştirilebilir hata raporlaması.

name: OrkhonJS CI
on: [push, pull_request]
jobs:
  kontrol:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: OrkhonJS Lint & Derleme
        uses: ./extensions/github-actions
        with:
          source-dir: './src'
          strict: 'true'
CI/CD Lint Derleme
🎨

Prettier Eklentisi

OrkhonJS dosyalarını otomatik biçimlendirme. Girinti boyutu, import sıralama, noktalı virgül tercihi, süslü parantez stili ayarları.

npm install --save-dev prettier-plugin-orkhonjs

// .prettierrc
{
  "plugins": ["prettier-plugin-orkhonjs"],
  "orkhonIndentSize": 4,
  "orkhonSortImports": true
}
Prettier Formatter Auto-format
📎

Snippet'ler Paketi

Tüm popüler editörler için OrkhonJS snippet koleksiyonu: VS Code, Vim UltiSnips, Emacs yasnippet, Sublime Text ve Atom. 20+ hazır şablon.

# Vim UltiSnips
cp extensions/snippets-pack/vim-ultisnips/orkhon.snippets \
   ~/.config/nvim/UltiSnips/

# Emacs
cp -r extensions/snippets-pack/emacs/* \
   ~/.emacs.d/snippets/orkhon-mode/

# Sublime Text
# Packages/User/ dizinine kopyalayın
Vim Emacs Sublime Atom

Dağıtım Adaptörü

Vercel ve Netlify'a Türkçe CLI ile dağıtım. Yapılandırma oluşturma, derleme, dağıtma ve yerel önizleme komutları.

npm install orkhonjs-deploy

# Yapılandırma oluştur
npx ork-dağıt yapılandır

# Vercel'e dağıt
npx ork-dağıt dağıt --hedef vercel --token TOKEN

# Yerel önizleme
npx ork-dağıt önizle
