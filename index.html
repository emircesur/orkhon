Neden OrkhonJS?

Modern geliştirme için ihtiyacınız olan her şey, Türkçe.

🇹🇷

Tamamen Türkçe

Anahtar kelimeler, tip isimleri, standart kütüphane — her şey Türkçe. tanım, işlem, sınıf gibi doğal anahtar kelimeler.

📝

Statik Tip Sistemi

Metin, Sayı, DoğruYanlış ile güçlü tipleme. Jenerik tipler, arayüzler ve tip çıkarımı.

⚛️

O-JSX (React)

React bileşenlerini tamamen Türkçe yazın. JSX sözdizimi ile modern UI geliştirme.

🎯

Çoklu Hedef

JavaScript, TypeScript, WASM, LLVM-IR, Kotlin, Swift, C++, Rust — tek kaynak, çok hedef.

🔧

Zengin Araç Seti

VS Code, JetBrains, Vim/Neovim eklentileri. Prettier, GitHub Actions, DevTools desteği.

🚀

Kolay Dağıtım

Vercel ve Netlify'a Türkçe CLI ile dağıtın. ork-dağıt dağıt --hedef vercel

Anahtar Kelime Haritası

OrkhonJS anahtar kelimeleri ve JavaScript karşılıkları

Değişkenler

tanım→let
sabit→const
değişken→var

İşlemler

işlem→function
dön→return
asenkron→async
bekle→await

Kontrol Akışı

eğer→if
yoksa→else
döngü→for
iken→while

Sınıflar

sınıf→class
uzatır→extends
yapıcı→constructor
yeni→new

Modüller

içe_aktar→import
dışa_aktar→export
kimden→from

Tipler

Metin→string
Sayı→number
DoğruYanlış→boolean
Boş→void
